## Recursos por categoria

| Categoria                       | Endpoints                                       |
| ------------------------------- | ----------------------------------------------- |
| **DICT**                        | Lookup de chaves Pix                            |
| **QR Code**                     | Decodificação de BR Code                        |
| **Charges**                     | Cobranças Pix (criar, listar, consultar, refresh) |
| **Payouts**                     | Pagamentos enviados (via chave ou QR)           |
| **Movements**                   | Extrato, saldo, movimentos                      |
| **Account**                     | Atalhos para a conta da API key                 |
| **Pix accounts**                | Tarifas, limites, rate limit por conta          |
| **Inbox**                       | Pix recebidos                                   |
| **API keys (per-account)**      | Gerenciamento de keys                           |
| **Outbound webhooks**           | Endpoints de webhook                            |

## Headers especiais

| Header                | Quando usar                                                     |
| --------------------- | --------------------------------------------------------------- |
| `Authorization`       | Sempre. `Bearer <api-key>`                                      |
| `Idempotency-Key`     | Em todo POST. 8–64 caracteres                                   |
| `X-Pix-Account-Id`    | Quando a key não está vinculada a uma conta única               |
| `Content-Type`        | `application/json` em requests com body                         |

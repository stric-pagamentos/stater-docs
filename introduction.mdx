A **Stric Platform** é uma API Pix que cobre o ciclo completo de pagamentos
instantâneos: envio e recebimento de Pix, criação de cobranças com QR Code,
consulta de titular pela chave Pix, gestão de contas, conciliação via extrato e
notificação em tempo real por webhooks.

## O que você precisa saber antes

<AccordionGroup>
  <Accordion title="Autenticação por API key" icon="key">
    Todas as requisições usam `Authorization: Bearer <sua-key>`. Veja
    [Autenticação](/guides/authentication).
  </Accordion>
  <Accordion title="Valores monetários em centavos (string)" icon="dollar-sign">
    R$ 10,50 é enviado como `"1050"`. Sempre string, nunca float — evita
    erros de precisão. Veja [Formato monetário](/guides/money-format).
  </Accordion>
  <Accordion title="Idempotência obrigatória nos POSTs" icon="rotate">
    Toda operação de escrita exige o header `Idempotency-Key`. Repetir a
    mesma chave retorna a resposta original. Veja [Idempotência](/guides/idempotency).
  </Accordion>
  <Accordion title="Multi-conta via header" icon="users">
    Se sua API key não está amarrada a uma conta específica, envie
    `X-Pix-Account-Id` indicando sobre qual conta a operação atua.
  </Accordion>
</AccordionGroup>

## URL da API

A base é **`https://stater.stric.io`**.

<Tip>
  Use sempre a variável `STRIC_BASE_URL` nos exemplos para facilitar migração de
  ambiente e evitar hardcode da URL nas integrações.
</Tip>
